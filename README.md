# ToDoアプリ

## 概要
本アプリは、日常のタスク管理を目的としたシンプルなToDoアプリです。  
HTML / CSS / JavaScriptのみを使用し、フレームワークに依存せずに実装しています。

タスクの追加・編集・完了管理・フィルター・並び替えなど、  
実用的な機能を段階的に実装しました。

---

## 機能一覧

### タスク管理機能
- タスクの追加
- タスクの編集
- タスクの削除
- ワンクリックで完了 / 未完了の切り替え

### 表示・操作機能
- 「すべて / 未完了 / 完了」のフィルター表示
- ドラッグ＆ドロップによるタスク並び替え
- 操作しやすさを意識したUIデザイン

---

## 工夫した点
- JavaScriptのイベント処理を整理し、可読性を意識した実装
- 状態（完了・未完了）に応じた表示切り替え
- 機能追加を段階的（Lv1〜Lv3）に行い、拡張しやすい構造に設計
